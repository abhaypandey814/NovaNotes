# NovaNotes
A modern, full-stack notes management application built with React.js, Node.js, Tailwind CSS, and MySQL (optional). NovaNotes allows users to create, edit, organize, and search notes with a clean, responsive interface and seamless backend connectivity.

🚀 Features

✨ Create, edit, delete, and manage notes
🔍 Search and filter notes by title or content
💾 LocalStorage support for instant UI updates
🗂️ Optional Node.js + MySQL backend for persistent storage
📱 Fully responsive UI using Tailwind CSS
⚡ Fast, modular, component-driven React architecture
🔄 Auto-sync behavior between LocalStorage and backend
🎨 Clean, minimal, user-friendly design

🛠️ Tech Stack
1. Frontend (React)
     React.js (Hooks)
     Tailwind CSS
     Axios
     React Router

2. Backend   
     Node.js
     MySQL
